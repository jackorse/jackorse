# Hi there! 👋 I'm Giacomo Orsenigo

**Computer Science Engineer (M.Sc. @ Polimi)**

🎓 I recently graduated in Computer Science and Engineering (M.Sc.) from **Politecnico di Milano**. 
I am deeply passionate about designing and developing complex, scalable, and resilient systems, with a strong focus on distributed architectures and Cloud/Edge technologies.

🔭 I recently completed my thesis on **Ermes**, a lightweight, stateful FaaS framework tailored for the Edge-Cloud Continuum, leveraging **WebAssembly** and **Go**.

---

### 🏆 Featured Projects

* ☁️ **[Ermes-thesis](https://github.com/ermes-thesis)**: A stateful FaaS platform optimized for edge computing environments. It executes serverless functions with high performance using WebAssembly and Redis. *(Go, Wasm, Redis)*
* 🌐 **[HACOGroupChat](https://github.com/jackorse/HACOGroupChat)**: A fully distributed, peer-to-peer chat application featuring causal message ordering and high availability under network partitions. *(Java, UDP, Docker)*
* 🎲 **[MyShelfie](https://github.com/jackorse/my-shelfie)**: A distributed Client-Server multiplayer game featuring concurrent match management and disconnection resilience. *(Java, Sockets, RMI)*
